# Headstart

## **Overview**

The headstart to develop project

## **Prerequisite**

- Terraform version `1.10.5`

- Python version `3.12.x`

- Package manager `uv`

- Version control `git`

- Build tool `make`

- GitHub CLI `gh`

- gcloud CLI `gcloud`

## **Account**

- Google Cloud Platform (GCP) account
